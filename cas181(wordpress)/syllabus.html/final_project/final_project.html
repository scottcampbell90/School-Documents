Final Project

Requirements

You will create a WordPress installation for your Final Project. Use your the name mysite for your site installation (yourdomain name/mysite). If you are going to keep this site hosted at this hosting location after the class is over, install the site at the root (do not create a directory for it).

Your site MUST include the following:

  1. Updated to 3.8
  2. A home (index) page.
  3. A Blog page. It must be named Blog in the navigation.
  4. A Contact page.
  5. A minimum of two and a maximum of four additional pages.
  6. The Blog must have Posts on it.
  7. The Home (index) page must have static content on it. You can put Posts as well if you choose but there must be static content.
  8. You must have a menu (navigation bar) with links to all main pages.
  9. All Posts must have at least two Tags and one Category.
  10. Theme - you may use Twenty Thirteen or Twenty Twelve  but you must customize the theme if you use them. OR - use a different theme.
  11. A minimum of two default widgets in either the sidebar or footer areas.
  12. A minimum of two custom widgets (copy and paste HTML code or use a Plugin as content).
  13. A slideshow or rotating pictures Plugin (a plugin for Flash content can be used in place of the slideshow Plugin).
  14. A Form plugin.
  15. Three Plugins of your choice. A Plugin can be used in a widget.
  16. A user login Plugin.
  17. Three additional users (not including you as the Admin). You must add the users and at least one must have the Contributor role. 
  18. A password protected (user restricted) area of the site. This does not include linking to an outside site that is password protected. It must be within your site.
  19. One social media feed (Twitter, Facebook, or Pinterest for example). You can use any of Landscape or Yarnstore social media feeds if necessary.
  20. One embedded video.
  21. One linked audio.
  22. At least 4 images.
  23. An Image Gallery on one page (the gallery can be in a post on a page).
  24. Content: Headings must be real text and reflective of your site subject. Post and Page content can be Lorem Ipsum.
  25. Admin area: Time Zone must be West Coast (or the time zone reflective of where your site is), your PCC e-mail address must be used, the Tagline must be changed from the default of the theme.
  26. Permalinks setting changed to something other than the default.
  27. It must have a Child Theme installed and the Child Theme must be the active theme (see the Module 5 lecture for instructions on installing a Child Theme plugin (One Click Child Plugin). The exception to this is if you use an Artisteer-generated theme. Then you do not need to have a child theme installed.

To submit your Final Project, email your instructor in Desire2Learn the link to your Final Project site.

  • In the message, list the link for the Final Project site and the admin username and password.
